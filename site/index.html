A memory layer for learning — not a chat log.

Learning Memory doesn't summarize the page for you. You write the rough thought first — even a fragment — and it builds on exactly that, grounded in what you're reading.

View source on GitHub → How to install

the idea

"AI should never replace your first thought." Most AI tools fix scattered learning by auto-summarizing everything — so you never actually engage with the material. This does the opposite: the AI only acts after you start writing.

what it does

Verified, not aspirational

Every feature below is implemented and tested working — not a roadmap.

01 — capture

Context-aware notebook

Captures the page title and any text you've highlighted automatically when you start a session — no manual copy-pasting.

02 — expand

Grounded AI expansion

Gemini reads your rough note, the page, and your highlight, then returns a beginner explanation, technical detail, example, and related concepts — built on what you wrote.

03 — honesty

Pushes back when you're off-topic

If your note has nothing to do with the page, it says so directly instead of forcing a connection — a distinct callout, not a normal expansion.

04 — recall

Recall banner

When a new session's concepts overlap something you explored days ago, it surfaces that past session by name — automatically, without you searching for it.

05 — map

Connections view

Groups every session by shared concept tag into a map of related learning, instead of a flat, forgettable list.

06 — privacy

Local-only by default

Runs entirely in chrome.storage.local — no account, no backend. Only the page title, your note, and highlight reach Gemini — never the URL.

try it

Loaded locally in under a minute

This is a Chrome extension, not a hosted web app — it runs inside your browser, so there's no URL to visit it at.

  1. Clone the repo.
    git clone https://github.com/tiemytime/memoir.git
  2. Install and build.
    pnpm install && pnpm build
  3. Load it unpacked. Open chrome://extensions, enable Developer mode, click Load unpacked, select the dist folder.
  4. Add your API key. Click the extension icon → Open Workspace → Settings, paste a free key from aistudio.google.com/apikey.
  5. Optional: seed demo data. Click Load demo data in Settings for three past sessions, so recall has something to find immediately.
